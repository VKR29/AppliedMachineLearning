# Assignment 3 — Testing & Model Serving

This repository implements a **spam text classifier scoring pipeline** along with **unit tests**, **Flask model serving**, and **coverage reporting**.

The project demonstrates how a trained machine learning model can be packaged into a reusable scoring function, tested using `pytest`, and exposed through an API endpoint.

---

## Features

* Score a trained model on input text using a reusable function
* Comprehensive unit tests for correctness and edge cases
* Flask API for serving predictions
* Integration testing of the API
* Test coverage report
