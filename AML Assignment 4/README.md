# Assignment 4 — Containerization & Continuous Integration
This repository contains a **machine learning scoring pipeline** for spam text classification along with **unit tests**, **Flask API serving**, and **Docker support**.

## Contents

* `score.py` — Core function that generates prediction and probability
* `app.py` — Flask API to serve the model
* `model.pkl` — Trained classification model
* `test.py` — Unit and integration tests using pytest
* `Dockerfile` — Container setup for deployment
* `requirement.txt` — Project dependencies
* `coverage.txt` — Test coverage report


This project demonstrates how to package a trained ML model, test it, and deploy it as a simple API using Docker.
